Outcome: Student can create, parse, and manipulate collections of objects.


1. In Game, create an array to hold all of the CoolThing objects.

2. In Game.spawn(), add the new CoolThing object to the array.

3. In Game.updateThings(), iterate through the array, updating each CoolThing.

4. In Game.updateThings(), while iterating, remove "dead" CoolThings from the display tree and array
